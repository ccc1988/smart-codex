# learn — 轻量复盘入口（A：实用优先）

## 定位
- 只做“轻量复盘”
- 不写代码、不跑命令、不改配置
- 仅追加写入知识库文件

## 触发
- 用户输入以 `learn:` 开头（如：`learn: 今天这个需求的复盘`）

## 输出（对话中）
固定三块，总计不超过 10 行：
1) 背景与目标（1–2 句）
2) 关键判断 / 踩坑点（2–4 条）
3) 下次更快的做法（2–4 条）

并同时给出：
- 写入预览（知识库文件前 20 行）
- 写入路径

## 写入规则（仅追加）
- 默认文件：`~/.codex/kb/one-man-playbook.md`
- 目录或文件不存在则创建
- 只追加（append），不修改历史内容
- 每次追加需带日期（本地日期，格式 `YYYY-MM-DD`）

## 追加格式（示例）
```
## YYYY-MM-DD
- 背景与目标：……
- 关键判断 / 踩坑点：
  - ……
  - ……
- 下次更快的做法：
  - ……
  - ……
```

## 边界
- 不影响 go / quote / fix / ship 的任何行为
- 不读取或写入其他路径
