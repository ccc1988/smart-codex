# Skill: report

## Workflow
1) Restate scope (project + period).
2) List completed items (3-6).
3) List in-progress items (3-6).
4) Highlight risks/blockers (0-3).
5) List next steps (3-6).

## Style
- Short bullets.
- Neutral tone.
- Avoid promises.

## Output format
- Summary → Done → In-progress → Risks → Next.

## Checklist
- 输入完整
- 计划最小
- 验证最小
- 结论明确

## 输出约束
- 不臆测
- 不夸大
- 不隐去关键限制
